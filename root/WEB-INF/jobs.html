Different Types of Fields in Computer Science

Job Title Job Description Median Annual Salary Projected Growth
Software Applications Developer
  • Analyze user's needs and design and develop software to meet those needs.
  • Collaborate with other company stakeholders to design and enhance the program or application.
  • Oversee the work of computer programmers, technologists, and technicians to ensure projects are running smoothly.
 $98,260 14 Percent
Computer Support Specialist
  • Answer questions from users about computer equipment or software.
  • Identify and solve software or hardware application problems in-person or via phone or email.
  • Set up equipment, install programs, and carry out minor repairs to hardware.
 $48,620 12 Percent
Computer Systems Analyst
  • Merge business and IT initiatives.
  • Analyze data processing problems to improve computer systems, enhance system compatibility, and develop procedures and quality standards.
  • Consult with business leaders to determine the role of the IT system.
 $85,800 14 Percent
Web Developers
  • Design, build, and maintain websites for clients and companies.
  • Write or design website content and perform site updates and other maintenance.
  • Work closely with the client or internal team to prioritize needs, develop content, and identify solutions.
 $64,970 14 Percent
Network Systems Administrators
  • Keep an organization's computer network up to date and running smoothly.
  • Maintain and administer computer networks, hardware, and software.
  • Perform data backups or recovery and troubleshoot issues when needed.
 $77,810 7 Percent
Computer Systems Engineers
  • Create solutions to complex applications problems, system administration issues, or network concerns.
  • Connect and communicate with clients about system needs and collaborate with software developers to identify appropriate design solutions.
  • Provide advice on project costs, design concepts, or design changes.
 $85,240 3 Percent
Database Administrators
  • Work as part of a project team to coordinate database development.
  • Write and code logical or physical database descriptions, specifying user access levels for each segment of the database.
  • Test programs and databases, correct errors, and make necessary modifications.
 $81,710 11 Percent
Software Quality Assurance Engineers
  • Document software defects, track bugs, and report defects to software developers.
  • Develop testing programs that address software scenarios, participate in product design reviews, and provide input on program functions.
 $85,240 3 Percent
